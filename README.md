# IT Helpdesk & Desktop Support Simulation

## Project Description
This project simulates a real-world IT Helpdesk and Desktop Support environment. It allows logging, tracking, and resolving common IT issues related to software, hardware, network, and printers.

## Features
- Log new IT support issues
- View all logged issues
- Mark issues as resolved
- Maintain issue history in CSV file
- Desktop support troubleshooting checklist

## Tools & Technologies
- Python
- CSV File Handling
- Basic IT Support Concepts
- Documentation using Markdown

## How to Run
1. Install Python
2. Download the project folder
3. Run:
   python helpdesk_simulation.py

## Use Case
This project simulates how Level 1 IT Support teams manage user-reported issues and track their resolution.

- Issue documentation
- Basic automation for IT operations
